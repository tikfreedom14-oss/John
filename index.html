01

ภาพรวมแผน

📊 วิเคราะห์จากมาราธอน Buriram 5:40:34

Johnnee จบมาราธอนมาแล้ว — มี base ดี ไม่ใช่มือใหม่. Split เต็ม 42km โชว์: 30km แรกคุมเพซดี (7:06-7:31) แล้วค่อยช้าลงช่วง 35-42km (7:47→8:04) ซึ่งเป็นเรื่องปกติของมาราธอน ไม่ได้พังยับ. Half 2:27:55 ทำนายมาราธอนได้ถึง ~5:08 = ศักยภาพ Sub 5:00 มีจริง. กลยุทธ์: ซ้อมด้วยเพซ Sub 5:15 (MP 7:27) · วันแข่งประเมินที่ km 25-30 · ถ้าแรงเหลือค่อยเร่งหา Dream 5:00.

⏱️ หัวใจของแผนนี้ — long run คุมด้วย "เวลา" ไม่ใช่ "ระยะ"
  • เพซ long run ของ Johnnee ~7:58-8:20 → 32km = 4 ชม. 6 นาที บนขา (นานเกินไปมาก!)
  • เพดานปลอดภัย = ไม่เกิน 3:30 ชม. · แผนนี้จบที่ 26km ≈ 3:27 ชม.
  • ทุก long run แสดงเวลาจริง · ถ้าวิ่งเกิน 3:30 ชม. ให้หยุด แม้ยังไม่ครบระยะ
  • เพิ่ม "คุณภาพ" (ใส่ MP ท้าย long) แทนการเพิ่มระยะ — ตรงกับจุดที่ต้องแก้พอดี
⚠️ อ่านก่อนเริ่ม
  • สัปดาห์แรก ~34 km/wk · peak สูงสุด 56 km/wk (W11, W14)
  • ถ้าตอนนี้วิ่ง ต่ำกว่า 28 km/wk → ลด W1-W3 ลง 20% แล้วไล่ตามตารางที่ W5
  • เพิ่ม volume ไม่เกิน +10%/สัปดาห์ — คุมไว้แล้วทุกสัปดาห์
  • Cut-off rule: เจ็บ >3/10 หรือเจ็บแหลมเฉพาะจุด (ข้อ/เอ็น/กระดูก) → เปลี่ยน quality/long วันนั้นเป็น Easy สั้นหรือพักทันที · ตารางคือแนวทาง ไม่ใช่คำสั่ง

โครงสัปดาห์: จันทร์พัก · อังคาร speed · พุธ easy · พฤหัส tempo/MP · ศุกร์ strength · เสาร์ easy+strides · อาทิตย์ long run. ยึด 3:1 เคร่งครัด (หนัก 3 · ลด 1).

เฟส 1 · Base

สร้างฐาน · long 12→16km · ฝึก วินัยวิ่งช้า (Easy ต้องช้าจริง) · tempo สั้น.

W5-7 (+ deload W8)

เฟส 2 · Build

เริ่ม MP run ที่เพซใหม่ 7:27 · long ถึง 21km (2:50 ชม.) · volume แตะ 50 km/wk.

W9-11 (+ deload W12)

เฟส 3 · Peak 1

long 22→24km · MP run 8-10km · เริ่มใส่ MP ท้าย long — ฝึกตรงจุดที่เคยช้าลง.

W13-15

เฟส 4 · Peak 2

🎯 W14 = long ไกลสุด 26km (≈3:27 ชม.) last 8km MP + ซ้อม fueling เต็มรูปแบบ — 4 สัปดาห์ก่อนแข่ง.

W16-17

เฟส 5 · Taper

ลด volume 56 → 39 → 29 km/wk. รักษาความคุ้นเพซ MP · ขาสดขึ้นทุกวัน.

W18 · 9-15 พ.ย.

เฟส 6 · Race Week

วิ่งเบาสั้นๆ + strides · ศุกร์-เสาร์พักเต็ม ไม่ยกเวท · Carb-load · อาทิตย์ 15 พ.ย. = RACE DAY 🏆

02

โซน & เพซ

คำนวณจากเป้าซ้อม Sub 5:15 (MP 7:27) และ Half PR 7:00/km. สำคัญสุดสำหรับ Johnnee: Easy ต้องช้าจริง — วินัยวิ่งช้าคือพื้นฐานของการคุมเพซวันแข่ง.

❤️ Heart Rate Zones

ยังไม่มีค่า Max HR / LT จริง — ใช้ % ของ Max HR + talk test ไปก่อน.

โซน% Max HRความรู้สึก (talk test)ใช้กับ
🔬 หาค่า HR จริง — อย่าใช้สูตร 220 − อายุ
  1. สูตร 220 − อายุ คลาดเคลื่อนได้ ±10-12 bpm → zone ผิดทั้งชุด
  2. ถ้ามีนาฬิกา Garmin: ทำ Lactate Threshold test → ได้ LT bpm + LT pace จริง
  3. ระหว่างนี้เชื่อ talk test — Easy = คุยได้เป็นประโยคยาว
03

ตาราง 18 สัปดาห์

กดแต่ละสัปดาห์เพื่อดูครบ 7 วัน. Long run (เวลาบนขา): 12→14→16→10→18→20→21→14→22→23→24→16→24→26 (3:27 ชม.)→20→16→10→🏆 · ตัวหนา = deload/longest · ทุก long run ≤ 3:30 ชม.

04

วิธีแล่นแต่ละวัน

ทุก session: warm-up jog 10 นาที → เนื้อหลัก → cool-down jog 5-10 นาที + ยืดเหยียด.

05

Race Pacing · เป้าคู่

วันแข่ง อาทิตย์ 15 พ.ย. 2026. ออกตัวที่เป้า B แล้วประเมินที่ km 25-30 — ถ้าแรงเหลือค่อยเร่งหา Dream.

🏁 กลยุทธ์วันแข่ง — เป้าคู่ (5:15 → Dream 5:00)
  1. km 1-15: ล็อกเพซ 7:27 ให้นิ่ง · อย่าเผลอเร็ว (ครั้งก่อนคุม 30km แรกได้ดีอยู่แล้ว)
  2. km 16-25: รักษาเพซ · เดิน 20-30 วิ ทุกจุดน้ำ ดื่มให้ครบ (ฝึกมาแล้วจากซ้อม) · Gel ทุก 30 นาที (เป้า 45-60g คาร์บ/ชม.) + เกลือแร่ · ประเมินร่างกาย
  3. km 25-30 = จุดตัดสินใจ: ถ้าแรงเหลือ + หายใจคุมได้ → เร่งเป็น 7:06 หา Dream 5:00
  4. km 31-42: จุดที่ครั้งก่อนช้าลง (7:47→8:04) — คราวนี้ฝึกมาแล้ว อดทนรักษาเพซ
  5. ถ้าคุม 5:15 ได้ = ทำลาย PR 25 นาที · ถ้าเร่งได้ = แตะ 5:00 · ร้อนเกินถอยเป้า A ก็ยังยอดเยี่ยม
06

เคล็ดลับ

⏱️ เวลา > ระยะ

Long run สิ่งที่ร่างกายรับรู้คือ เวลาบนขา. 26km ที่เพซ Johnnee = 3:27 ชม. เท่ากับ 38km ของนักวิ่งเร็ว. อย่าเทียบระยะกับคนอื่น — เกิน 3:30 ให้หยุด.

🎯 คุมเพซ 30-42km

มี base มาราธอนแล้ว! จุดพัฒนาคือช่วง 35-42km ที่เคยช้าลง. ฝึกใส่ MP ท้าย long run + MP practice บ่อยๆ + run-walk เป็นระบบทุกจุดน้ำ.

🎯 เป้าคู่ 5:15 / 5:00

ซ้อม MP ที่ 7:27 ให้ชิน. Half ทำนายได้ถึง 5:08 = Dream 5:00 มีลุ้น! ถ้าซ้อมไหว ลองแตะ 7:06 บ้างใน MP run ช่วง Peak.

🐢 Easy ต้องช้าจริง

Easy run ที่ 8:05-8:30 — ช้าจนคุยเป็นประโยคยาวได้. ฝึกวินัยวิ่งช้า = พื้นฐานของการคุมเพซวันแข่ง. อย่าให้อัตตาพาไป.

🍚 Fueling

Long run 90 นาที+ Gel ทุก 30 นาที (เป้า 45-60g คาร์บ/ชม.) + เกลือแร่ (อย่ารอหิว). ฝึกกินในซ้อมให้ท้องชิน — W14 คือการซ้อมใหญ่. Carb-load 2-3 วันก่อนแข่ง.

💪 Strength (ศุกร์)

Squat · Lunge · Glute Bridge · Core · Calf Raise. posterior chain รักษาฟอร์มตอนล้า km 30+. สัปดาห์แข่งงดทั้งหมด.

07

Recovery Budget · ปรัชญาการซ้อม

3 หลักที่ใช้ตัดสินใจทุกวัน: Easy ต้องเบาจริง · Recovery คืองบก้อนเดียว · ไม่ต้องซ้อมจนหมดแรง. ตัวเลขบอกครึ่งหนึ่ง ความรู้สึกบอกอีกครึ่ง.

✅ เช็ค 4 อย่าง — Easy วันนี้เบาพอไหม

1. Heart Rate — อยู่ในโซน Easy (Zone 2) ไม่ไต่ขึ้นเรื่อยๆ
2. Pace (8:05-8:30) — ร้อน / นอนน้อย / ล้าสะสม → เพซที่ช้าลงคือเพซที่ถูกต้องของวันนั้น
3. RPE 3-4/10 — ยังผ่อนคลาย พูดเป็นประโยคได้ ไม่ฝืน
4. การหายใจ — หายใจทางจมูกได้สบาย ลึก ต่อเนื่อง

4 อย่างไปทางเดียวกัน = Easy ทำหน้าที่จริง · ขัดกัน = เชื่อ HR และร่างกาย

💰 Recovery Budget — งบก้อนเดียว

การนอน · ความเครียดงาน · อาหาร/น้ำ/เกลือแร่ · load สะสม — ทั้งหมดแบ่งใช้งบฟื้นตัวก้อนเดียวกัน. ถ้าความสามารถฟื้นตัวลดลง Easy Run เดิมก็ไม่ใช่ Easy Run อีกต่อไป แม้เพซเท่าเดิม.

⭐ วิธีวัดว่าพอหรือยัง: ไม่ได้วัดจากวันนั้น แต่วัดจาก quality session ครั้งถัดไป (MP 7:27 / Tempo 6:50-7:05 / Long run) — ทำได้ตามเป้า = ที่ผ่านมาเหมาะสม · ทำไม่ได้ = ต้องพักเพิ่ม

🏋️ ไม่ต้องซ้อมจนหมดแรง (RIR 1-3)

หยุดก่อนหมดแรง 1-3 ครั้ง ให้ความแข็งแรงใกล้เคียงกัน แต่ความล้าสะสมต่ำกว่ามาก. เป้าของเราไม่ใช่ยกเวทให้เก่ง แต่คือรักษาคุณภาพโปรแกรมวิ่งทั้งสัปดาห์.

ถามก่อนทุก session: ต้องการ adaptation อะไร · จำเป็นต้องหมดแรงจริงไหม · recovery ที่จ่ายคุ้มไหม · จะกระทบการวิ่งใน 1-3 วันข้างหน้าไหม

🐢 วินัยวิ่งช้า = จุดที่ต้องฝึกที่สุด

Easy ที่ 8:05-8:30 อาจรู้สึกช้าจนอยากเร่ง — แต่นั่นคือเพซที่ถูกต้อง. จุดที่ต้องพัฒนาคือช่วง 35-42km ที่เคยช้าลง ซึ่งต้องใช้ขาที่สดในวัน Long run ถ้าใช้งบฟื้นตัวหมดไปกับ Easy ที่เร็วเกิน วัน Long ก็จะฝึกคุมเพซปลายไม่ได้

⚠️ กับดักที่เจอบ่อยสุด
  1. Easy เร็วเกินวันนี้ → ไม่รู้สึกอะไรเลย แต่ 2 วันถัดมา tempo ทำไม่ครบ / long run หนักผิดปกติ
  2. แล้วเข้าใจผิดว่า "ซ้อมหนักไม่พอ" ทั้งที่จริงคือ "ความล้าที่เกิน" ไม่ใช่ "สิ่งที่ขาด"
  3. เพซช้ากว่าเป้าแต่ HR สูงกว่าเป้า = สัญญาณล้าสะสม ไม่ใช่ "วันฟอร์มไม่ดี"
  4. ไม่มีวันเบาจริงหลายวันติด = เสี่ยง overreach แม้แต่ละวันดูไม่หนัก
🌱 มุมที่ควรจำไว้
  1. ร่างกายไม่ได้เปลี่ยนจากการซ้อมยอดเยี่ยมครั้งเดียว แต่จากการซ้อมที่ "ดีพอ" ซ้ำแล้วซ้ำอีก เป็นเดือน
  2. โปรแกรมทำไม่ครบ / ย้ายวัน / ตัดบางเซสชัน — เป็นเรื่องปกติของคนที่มีชีวิตจริง ไม่ใช่ความล้มเหลว
  3. อย่าให้คะแนนตัวเองจากสิ่งที่ยังทำไม่ได้ จนลืมมองสิ่งที่ทำได้แล้ว
  4. สัปดาห์ที่ไม่ดีที่สุด ≠ สัปดาห์ที่สูญเปล่า
