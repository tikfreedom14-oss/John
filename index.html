01

ภาพรวมแผน

📊 วิเคราะห์จากมาราธอน Buriram 5:40:34

Johnnee จบมาราธอนแล้ว มี base ดี! Split เต็ม 42km โชว์: 30km แรกคุมเพซดี (7:06-7:31) ค่อยช้าลงช่วง 35-42km (7:47→8:04) ซึ่งปกติของมาราธอน ไม่ได้พังยับ. เป้าที่ 2 คือพัฒนา ~20 นาที เป็น Sub 5:20. แผนเน้น: ดัน MP ให้ชินเพซใหม่ (7:35) + ฝึกคุมเพซช่วง 30-42km ที่เคยช้าลง.

แผน 18 สัปดาห์ 4 เฟส หลัก 3:1 (หนัก 3 · ลด 1). โครงสัปดาห์: จันทร์พัก · อังคาร speed เบา · พุธ easy · พฤหัส tempo/MP · ศุกร์ strength · เสาร์ easy+strides · อาทิตย์ long run.

เฟส 1 · Base

W1-5 · สร้างฐาน endurance · long run 12→20km · ฝึกวิ่งช้าสม่ำเสมอ

เฟส 2 · Build

W6-10 · เพิ่ม volume + MP · long run ถึง 28km · ฝึกคุมเพซ

เฟส 3 · Peak

W11-14 · long run ไกลสุด 30-32km · ซ้อม fueling + คุมเพซปลายทาง

เฟส 4 · Taper

W15-18 · ลด volume · รักษาความคุ้น pace · พร้อมแข่ง

02

โซน & เพซ

เพซจากเป้า Sub 5:20 (MP 7:35). Johnnee มี base มาราธอนแล้ว—เน้นดัน MP ให้ชินเพซใหม่ + long run คุม Zone 2.

03

ตาราง 18 สัปดาห์

กดแต่ละสัปดาห์ดู 6 วัน (จันทร์พัก). Long run ไต่: 12→14→16→12→18→20→22→16→24→26→28→18→30→32→taper. ตัวหนา = deload.

04

วิธีแล่นแต่ละวัน

ทุก session: warm-up jog 10 นาที + strides → เนื้อหลัก → cool-down + ยืด.

🏁 กลยุทธ์วันแข่ง — พัฒนาจาก 5:40
  1. km 1-15: ล็อกเพซเป้า 7:35 ให้นิ่ง อย่าเผลอเร็ว (ครั้งก่อนคุม 30km แรกได้ดีอยู่แล้ว)
  2. km 16-30: รักษาเพซ · ดื่มน้ำทุกจุด · Gel ทุก 40 นาที (จุดสำคัญกันช้าลง)
  3. km 31-42: จุดที่ครั้งก่อนช้าลง (7:47→8:04) — คราวนี้ฝึกคุมเพซมาแล้ว อดทนรักษา 7:35
  4. ถ้ารักษาเพซช่วง 35-42 ได้ = ทำลาย PR 5:40 แน่นอน
06

เคล็ดลับ

🎯 คุมเพซ 30-42km

Johnnee มี base มาราธอนแล้ว! จุดพัฒนาคือช่วง 35-42km ที่เคยช้าลง. ฝึกคุมเพซใน long run ช่วงท้าย + MP practice บ่อยๆ.

🎯 ดัน MP ใหม่

เป้าใหม่ 7:35 เร็วกว่าเพซมาราธอนเดิม (8:04). ต้อง MP practice บ่อยๆ ให้ร่างกายชินเพซใหม่ ก่อนวันแข่ง.

🍚 Fueling

Long run 90 นาที+ Gel ทุก 40 นาที. ฝึกกินในซ้อมให้ท้องชิน—สำคัญมากสำหรับป้องกัน fade.

💪 Strength

วันศุกร์: squat, lunge, glute bridge, core, calf. posterior chain ช่วยรักษาฟอร์มตอนล้า km 30+.

⏱️ คุมเพซ

ใช้นาฬิกาตั้ง pace alert—เตือนถ้าเร็วเกิน. ฝึกวิ่ง even pace ในซ้อม จะติดเป็นนิสัยวันแข่ง.

🛌 ฟื้นตัว

นอนพอ · deload สำคัญ · เจ็บให้พัก · สม่ำเสมอชนะหักโหม. มาราธอนแรก—สนุกกับการเดินทาง.
